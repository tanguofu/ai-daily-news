# AI Daily News 📰

每日热点内容聚合 - 自动化新闻订阅

## 📋 内容来源

- 💻 **Hacker News** - 技术热点
- 💾 **GitHub Trending** - 开源项目
- 📱 **Product Hunt** - 创新产品
- 🔥 **知乎热榜** - 热门话题
- 📝 **少数派** - 效率工具
- 🏀 **虎扑** - 体育资讯
- 💰 **华尔街见闻** - 财经要闻

## 🤖 自动化

由 OpenClaw 每日自动生成并推送

## 📂 目录结构

```
reports/
  ├── 2026-02-09.md
  ├── 2026-02-10.md
  └── ...
```

## 🔧 技术栈

- Python 3.11+
- Playwright (网页截图)
- OpenClaw (任务调度)
- GitHub Actions (可选)
